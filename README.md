# "What Really Happened" (WRH) 75-Session Master Curriculum
## The Complete System Logic Translation & Regulation Training Package

### **Overview**
The **75-Session Master Curriculum** is a proprietary, non-clinical psychoeducational intervention designed by **Capitol Contracts LLC**. It is a structured, replicable program for behavioral health, recovery, and risk mitigation, with a heavy emphasis on veterans. 

This curriculum frames trauma recovery as **System Logic Translation**—treating trauma responses as predictable engineering problems (mechanical loops, wiring patterns, and interruptible biological programs) rather than character flaws or emotional vulnerabilities.

### **Core Components**
This unified package integrates three foundational building blocks into one sequential 75-session flow:
1.  **Part I: The System Logic Map (Sessions 1–26)** – Based on the *26 Laws of Survival*.
2.  **Part II: The Core Operational Journey (Sessions 27–56)** – The original *30-Session Pilot Program*.
3.  **Part III: Advanced System Specialization (Sessions 57–75)** – The *19-Session Advanced Series*.
4.  **Part IV: Facilitator Standardization Toolkit** – Professional tools for standardized delivery.

### **Design Principles**
*   **Engineering Framing**: Shifting from "vulnerability-first" to neutral, technical diagnostics.
*   **Mechanism Breakdowns**: Every session applies system logic to observe behavior, translate the loop, and enable participant control.
*   **Full Facilitator Scripts**: Standardized scripts for non-clinical facilitators to ensure scalability and low re-traumatization risk.
*   **No Summarization**: All 75 sessions are preserved in their full, original detail to maintain IP integrity.
